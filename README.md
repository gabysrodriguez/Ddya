Estructuras de Datos y Diseño de Algoritmos

Este repositorio contiene el desarrollo de las actividades realizadas durante el curso, donde se trabajaron conceptos fundamentales de diseño de algoritmos, análisis de complejidad y estructuras de datos.
Cada carpeta corresponde a una semana del curso y contiene implementaciones en Python, así como diagramas y documentación del proceso de diseño.

El objetivo del repositorio es mostrar la evolución del aprendizaje a lo largo del semestre, desde evaluaciones diagnósticas iniciales hasta la implementación de técnicas avanzadas de resolución de problemas.

Organización del repositorio

El repositorio está dividido por semanas, donde cada carpeta incluye código, diagramas y documentación relacionados con los temas estudiados.

Semana 1 — Prueba técnica inicial

Durante la primera semana se realizó una prueba técnica diagnóstica con el objetivo de evaluar los conocimientos previos sobre programación y lógica algorítmica.

Las actividades incluyeron:

Resolución de problemas utilizando Python

Implementación de algoritmos básicos

Elaboración de diagramas de flujo

Evaluación de la capacidad para estructurar soluciones a problemas computacionales

Esta semana permitió establecer una base para el desarrollo de los temas posteriores del curso.

Semana 2 — Estrategia incremental y análisis asintótico

En esta semana se trabajó el enfoque de estrategia incremental, donde los algoritmos se construyen paso a paso agregando elementos a una solución parcial.

También se introdujo el concepto de análisis asintótico, utilizado para evaluar la eficiencia de los algoritmos en términos de tiempo de ejecución.

Las actividades realizadas incluyeron:

Desarrollo de laboratorios en Python

Diseño de diagramas de flujo formales

Redacción de historias de usuario

Análisis de complejidad temporal de algoritmos

Documentación estructurada del proceso de diseño

Semana 3 — Divide y vencerás y Teorema Maestro

En esta semana se estudió la técnica algorítmica de Divide y Conquistar, la cual consiste en dividir un problema en subproblemas más pequeños, resolverlos de forma independiente y luego combinar sus soluciones.

También se introdujo el Teorema Maestro, utilizado para analizar la complejidad de algoritmos recursivos.

El diseño formal de los ejercicios incluyó:

Implementación de algoritmos utilizando divide y conquistar

Análisis de complejidad usando el Teorema Maestro

Elaboración de diagramas de flujo

Creación de diagramas de casos de uso

Desarrollo de diagramas de secuencia

Implementación de soluciones en Python

Semana 4 — Programación dinámica

Durante esta semana se profundizó en la técnica de programación dinámica, utilizada para resolver problemas complejos mediante la reutilización de resultados previamente calculados.

Se trabajó principalmente con el concepto de memoización, que permite optimizar algoritmos recursivos evitando cálculos repetidos.

Las actividades incluyeron:

Implementación de algoritmos con memoización

Desarrollo de soluciones optimizadas

Aplicación de técnicas vistas en semanas anteriores

Diseño formal completo de los algoritmos

Documentación estructurada del proceso de solución

Esta semana consolidó los conocimientos sobre diseño y optimización de algoritmos.

Semana 7 — Estructuras de datos lineales

En esta semana se inició el estudio de estructuras de datos lineales, introduciendo un nuevo enfoque para resolver problemas mediante el uso de estructuras organizadas de datos.

Se trabajó principalmente con:

Pilas (Stacks)

Colas (Queues)

Las actividades incluyeron:

Implementación de estructuras lineales en Python

Resolución de problemas utilizando estas estructuras

Aplicación de todos los conceptos vistos previamente en el curso

Esta etapa marcó la transición hacia el uso de estructuras de datos para mejorar la eficiencia y organización de los algoritmos.

Tecnologías utilizadas

Python — implementación de algoritmos

Diagramas de flujo — representación del proceso lógico

Diagramas de casos de uso

Diagramas de secuencia

Análisis de complejidad algorítmica
